Sentiment Analysis Of Arabic Tweets using NLP

The project contains 2 Python Files:
    Normalize.py
    Main.py

To use these files, make sure to download the necessary libraries using the pip command in the terminal.

Order Of Execution:
-------------------
1. Normalize.py (optional and once for all)
2. Main.py (as much as you want)

Important Note: Make sure to not change the order of functions in "Normalize.py" file.
Use this command to be able to run the application
python -m nltk.downloader popular
